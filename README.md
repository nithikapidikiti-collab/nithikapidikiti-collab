## About

First-year **BIT Co-op Scholar at UTS**, incoming SWE intern at **Woolworths Group** (July 2026).

I like building things close to the metal. Most of my repos here are network systems written in C without libraries — a DNS resolver, HTTP server, VPN tunnel, TCP/IP stack, P2P file transfer, port scanner. I'm also interested in AI engineering and full-stack product work.

## Projects

All of these are built from scratch — no networking libraries.

| Project | What it does | Lang |
|---|---|:---:|
| [tcp-ip-stack](https://github.com/nithikapidikiti-collab/tcp-ip-stack) | Full TCP/IP stack — Ethernet, ARP, IP routing, TCP handshake + retransmission over macOS TUN | C |
| [tinyvpn](https://github.com/nithikapidikiti-collab/tinyvpn) | VPN tunnel with AES-256-GCM encryption, DH key exchange, UDP transport | C |
| [dns-resolver](https://github.com/nithikapidikiti-collab/dns-resolver) | Recursive DNS resolver parsing wire format manually, traces from root servers | C |
| [http-server](https://github.com/nithikapidikiti-collab/http-server) | Concurrent HTTP/1.1 server with POSIX sockets and fork-based concurrency | C |
| [p2p-fileshare](https://github.com/nithikapidikiti-collab/p2p-fileshare) | Peer-to-peer file transfer with chunked streaming directly over raw TCP | Python |
| [port-scanner](https://github.com/nithikapidikiti-collab/port-scanner) | Fast multithreaded TCP port scanner with thread pooling | Python |
